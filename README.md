# Data Analytics Portfolio

This repository contains data analytics dashboards created using Excel and Power BI.

---

## Jewellery Sales Dashboard (Excel)

### Problem Statement

Analyze jewellery store sales performance to identify top products, revenue trends, and city-wise sales distribution.

### Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers

### Dashboard Features

* KPI metrics (Total Sales, Cost, Profit, Profit Margin)
* Monthly sales trend
* Sales by category
* Top products by revenue
* Interactive filters

### Key Insights

* Necklaces generated the highest revenue.
* Sales peaked during festive months.
* Ahmedabad and Pune contributed the highest sales.
